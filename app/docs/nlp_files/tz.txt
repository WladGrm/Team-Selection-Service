Название
Система предназначена для автоматизации процесса формирования учебного плана.
Цель
С привлечением экспертных систем искусственного интеллекта автоматизировать процесс составления учебного плана, учитывая нагрузки на каждого преподавателя, трудоемкость учебных программ в неделю, удобное распределение нагрузки по дням недели для учащихся, разработав для этого информационную систему.
Объект
Объектом автоматизации является процесс составления учебного плана с учетом трудоемкостей программ обучения.
Содержание
В состав программного комплекса должны включаться следующие подсистемы: подсистема планирования занятий; подсистема графического отображения расписания в виде таблицы; подсистема администрирования; файловое хранилище домашних заданий; база данных учебных планов, база данных свободных часов преподавателей.
Технические требования
Информационный обмен между подсистемами должен осуществляться через единое информационное пространство и посредством использования стандартизированных протоколов и форматов обмена данными. Все компоненты подсистем АСУ должны функционировать в пределах единого логического пространства, обеспеченного интегрированными средствами серверов данных и серверов приложений
